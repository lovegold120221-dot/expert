## 6. TikTok Content Strategy

### Creative Triggers (from Top Ads Analysis)

| Trigger | Description | Execution |
|---------|-------------|-----------|
| **Before/After** | Direct narrative contrast | Show problem → solution transformation |
| **Soft Lifestyle** | Familiar everyday scenes | Product integrated into natural use cases |
| **Curiosity Hook** | Lead with intrigue | Open question or mysterious visual, then reveal |
| **Scenario Framing** | Make customization emotional | Specific use-case storytelling |
| **Immersion** | Close-up product focus | Detailed visual of product in action |
| **Trend-native** | Match current TikTok styles | Platform-specific editing, pacing, music |

### TikTok Video Best Practices

| Element | Recommendation |
|---------|---------------|
| **Length** | 15-30s for ads, 21-34s for organic |
| **Aspect Ratio** | 9:16 (vertical) — full-screen mobile |
| **First 3 seconds** | Hook — curiosity, question, or bold statement |
| **Captions** | Always include — majority watch without sound |
| **Music** | Use trending sounds or AI-generated |
| **Pacing** | Fast cuts, dynamic transitions every 2-4s |
| **CTA** | Clear, early, and repeated |
| **Native feel** | Match TikTok aesthetics — not polished production |

### Trend-Driven Content Creation Workflow

```
1. Analyze Top Ads in Creative Center → identify winning patterns
2. Research trending sounds and formats via Creative Center
3. Write script optimized for TikTok pacing (hook + body + CTA)
4. Generate video using Creative Studio or AI video models
5. Add captions, music, and effects
6. Use Recommended Creatives to generate variants
7. Deploy and monitor performance
8. Optimize with Symphony Automation during delivery
```
